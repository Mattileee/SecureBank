# SecureBanks Backend

This is the backend API server for the SecureBanks project.  
It handles user registration, login, authentication, and dashboard access using Supabase.

## Features

- ✅ User signup
- ✅ Email confirmation
- ✅ Secure login with JWT
- ✅ Protected dashboard route

## Tech Stack

- Node.js
- Express
- Supabase
- TypeScript
